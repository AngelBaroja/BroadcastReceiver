# BroadcastReceiver

# Descripción de la app
Aplicación Android desarrollada en Java que implementa un `BroadcastReceiver`
para escuchar el evento del sistema `Intent.ACTION_USER_PRESENT`, el cual se
dispara cuando el usuario desbloquea la pantalla del dispositivo.

Al detectar el desbloqueo, la aplicación realiza tres acciones:
- Muestra un Toast con el mensaje "Pantalla desbloqueada detectada".
- Registra un mensaje en Logcat indicando que ocurrió el evento.
- Abre el marcador telefónico del dispositivo con el número 2664553747 precargado.
